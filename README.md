# AI Language Translator

## Overview

AI Language Translator is a web-based application that translates text from one language to another using a translation API. The application allows users to select source and target languages, enter text, and receive translated output instantly.

## Features

* Translate text between multiple languages
* User-friendly interface
* Source and target language selection
* Real-time translation using API integration
* Swap language functionality
* Copy translated text to clipboard
* Responsive and modern UI

## Technologies Used

* HTML
* CSS
* JavaScript
* MyMemory Translation API

## Project Structure

## How It Works

1. User enters text in the input field.
2. User selects source and target languages.
3. The application sends the text to the translation API.
4. The translated text is returned and displayed in the output area.
5. Users can swap languages or copy the translated text.

## Supported Languages

* English
* Hindi
* Telugu
* French
* Spanish

## Sample Usage

Input:

Hello, how are you?

Source Language:

Target Language:
